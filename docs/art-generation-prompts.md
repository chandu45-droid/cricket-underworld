## SECTION 2 — ALIGNMENT VARIANTS
*Use these as ADDITIONS to any player prompt above when the player has a specific alignment*

### Alignment Visual Modifiers

**SPOTLESS alignment (+80 to +100):**
```
Add to prompt: "pure white aura behind subject, golden halo light from above, 
immaculate clean kit, noble heroic expression"
```

**CLEAN alignment (+20 to +79):**
```
Add to prompt: "soft blue rim light, crisp clean kit, honest determined expression, 
natural stadium lighting"
```

**GREY alignment (-19 to +19):**
```
Add to prompt: "neutral flat lighting, slightly worn kit, ambiguous expression — 
neither hero nor villain, watching eyes"
```

**CORRUPT alignment (-20 to -79):**
```
Add to prompt: "red and shadow split lighting — one side lit, one in darkness, 
slightly dishevelled kit, knowing smirk, dangerous eyes, shadows pooling around subject"
```

**DEEP CORRUPT alignment (-80 to -100):**
```
Add to prompt: "near-total darkness except for deep crimson underlighting from below, 
black kit with dark red accents, villain energy, cold calculating eyes without remorse, 
smoke or dark mist wisping around subject, mafia don aura"
```

| Tool | Best for | Notes |
