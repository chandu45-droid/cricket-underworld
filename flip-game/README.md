# FLIP — Gravity Runner (REJECTED)

**Status:** Rejected by founder  
**Reason:** Reflex/runner game, not strategic enough. User wants strategy + adventure, not twitch gameplay.

## What it was
- HTML5 Canvas gravity-flipping runner
- Tap to flip gravity, dodge obstacles
- Smoothstep difficulty curve, zone system
- Social features (seeded challenges, WhatsApp share)
- PWA with service worker

## Why it failed
- Too hard initially (fixed with gentler curve, but core concept rejected)
- User doesn't want reflex-based games at all
- No strategic depth — just reaction time
- Not adventurous — no exploration, discovery, or progression
